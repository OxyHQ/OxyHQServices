# Accounts (MyAccount)

The Oxy MyAccount app. Built on Expo Router + `@oxyhq/services`, it covers identity creation, key management, recovery, payment methods, and privacy settings.

<Badge variant="success">App</Badge> <Badge>Expo / RN / Web</Badge>

## Highlights

- 11 supported locales (English + Spanish fully populated).
- Typed Expo Router routes — no `as any` casts.
- Error boundaries at root, `(tabs)`, and `(auth)` layout levels.
- Mandatory recovery-phrase acknowledgement before identity creation completes.
- Activity history powered by `GET /security/activity`.
