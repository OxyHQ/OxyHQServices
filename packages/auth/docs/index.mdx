# Auth App

The standalone Vite app that powers `auth.oxy.so`. Handles sign-in, sign-up, account authorization, password recovery, and serves as a FedCM Identity Provider.

<Badge variant="success">App</Badge> <Badge>Vite + React</Badge>

## Highlights

- Multi-step login (identifier → password → 2FA).
- Per-step entrance animations and route-level fade transitions.
- Bloom color preset applied to CSS variables via `applyColorPreset()`.
- FedCM IdP endpoints served alongside the SPA.
