# Inbox

The Oxy email app. End-to-end encrypted keys stored as non-extractable `CryptoKey` in IndexedDB. Offline-first via TanStack Query's persist-client + Service Worker manifest precaching.

## Highlights

- Non-extractable `CryptoKey` for end-to-end encryption keys.
- Service Worker cache + manifest precache.
- Accessibility-labelled UI with Bloom typography.
